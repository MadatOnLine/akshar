## Overview

This document will contain the formal technical specification for the Akshar Protocol, including:

- Message format and encryption scheme
- Propagation algorithm and diversity scoring
- Hop-local traceability storage format
- Proof-of-humanity tier requirements and verification
- Token reward calculation and distribution
- Relay node protocol and message buffering
- ZK proof-of-origin scheme
- Node score computation
