# अक्षर · Akshar Protocol

**A Decentralized, Privacy-Preserving Communication Protocol with Native Economics**

> *Authentic ideas spread through diverse human networks. Manufactured consensus clusters. This protocol rewards the former and naturally slows the latter — without anyone reading your messages.*

---

## What is Akshar?

Akshar Protocol is an open, decentralized communication protocol that reimagines how ideas spread across human networks. It unifies two communication models — private group conversation and public broadcast — into a single protocol where sharing is attributed, privacy is architectural, and participants who create value are fairly compensated.

| Pillar | Description |
|---|---|
| **Privacy-First** | No keys held by anyone except the users themselves |
| **Economics-Native** | Good acts earn tokens — curation, origination, relay |
| **Censorship-Resistant** | No central server to shut down, no global graph to seize |
| **Accountable Sharing** | Every share is visible and attributed — no anonymous forwarding |

## Core Concepts

**Two-Layer Communication:** Private groups where ideas are born and debated. Public sharing where they earn their propagation.

**Proof-of-Curation:** Nodes earn $RSN tokens by performing genuinely valuable work — creating good ideas, curating them early, spreading them responsibly across diverse communities.

**Topological Diversity:** The propagation algorithm measures the structural diversity of spread — never message content. Content that bridges unconnected communities spreads fastest.

**Tiered Proof-of-Humanity:** Bot creation is made economically irrational at scale through compounding friction — device proof-of-work, human-evaluated puzzles, token burns, time gates, and social vouching.
